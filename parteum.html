Projetos

Bee Frutas

Bee Frutas é um jogo de tabuleiro digital que desenvolvi em Swing Java onde dois competidores entram em uma floresta repleta de frutas.

Farmaland

Neste jogo educativo desenvolvido com Java Swing, você assume o papel da farmacêutica Elloisa em um mundo industrializado onde infecções são uma ameaça grave.

MangueTrack

O Mangue Track é uma aplicação de rotas de ônibus, onde o usuário pode escolher o ponto de partida e o destino.

Arreda

Uma solução antifraude desenvolvida para prevenir o público idoso de golpes.
